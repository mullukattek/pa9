Fruit Ninja Game PA 9 (CptS 122)

Description:
This is a simple fruit ninja style game that we made in C++. 
The goal of the game is to click on fruits to slice them and earn points.
If the player clicks on a bomb, the game ends.

How to Play:
- Slice fruit to earn points
- Do not hit bombs
- In multiplayer mode, two players compete to get the highest score
- The game ends when the round is over or a losing condition happens

Bonus Feature:
The bonus feature in this project is multiplayer network play.
Two players can connect and play the game together using socket communication.
